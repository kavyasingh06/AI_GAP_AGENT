# AI Overview Content Gap Agent

## Overview

This project implements an **AI-powered agent** that analyzes content appearing in **Google AI Overviews** and compares it with a client's article to identify **content structure gaps and improvement opportunities**.

The agent automates the manual workflow used by SEO teams to understand **why certain articles are chosen as sources in Google AI Overviews** and what content improvements can help a client’s article perform better.

---

# Features

The agent performs the following steps:

1. **Search Google results for a keyword**
2. **Extract top source URLs** that may appear in AI Overview summaries
3. **Scrape content from those pages**
4. **Scrape the client article**
5. **Analyze structural differences using an LLM**
6. **Generate a structured content gap report**

# Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/ai-overview-gap-agent.git
cd ai-overview-gap-agent
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

# Assumptions and Design Decisions

* AI Overview URLs are approximated using top organic search results.
* BeautifulSoup is used for lightweight scraping.
* The first 5000 characters of page text are sent to the LLM to reduce token usage.
* JSON output was chosen for simplicity and automation.

---

# Limitations

* Google AI Overview sources may vary by location and time.
* Some websites block scraping or return incomplete content.
* The LLM analysis depends on prompt quality and available model capabilities.
